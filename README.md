## Prerequisites

- Require `Node version 18.x`

## Installing

To install Shop demo, follow these steps:

```
git clone https://github.com/minhduc-mll/internship-2023.git
```

```
yarn
yarn dev
```

## Features

- View product by category.
- View product detail.
- View new arrivals.
- View collections.
- Add to shopping cart.
- View shopping cart
- Checkout.
- Search product.
